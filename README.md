# Space Nft

> NFT marketplace for space-themed digital collectibles

## Features

- Mint NFTs
- Auction system
- Collection browsing
- Royalty management

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Solidity, Rust, Web3.js
- AI: MiMo V2 Flash, Claude Code

## Installation

```bash
git clone https://github.com/nimotvl/space-nft.git
cd space-nft
npm install
npm run dev
```
